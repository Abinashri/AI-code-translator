# Code Translator

A tool that converts Java code into Python using custom logic.

## 🚀 Features
- Converts print statements
- Converts loops (for, while)
- Converts if-else conditions
- Handles basic indentation

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Node.js (Backend)

## 📌 Example
Java: System.out.println("Hello");  
Python: print("Hello")

- Support more languages
